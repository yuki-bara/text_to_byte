# TEXT_TO_BYTE

Why would anyone want to convert text written with the numbers 1 and 0 into binary? I don't know either, and I don't know what it's used for. I just did it for fun.

# How to use the command

```bash
tts -fw byte -o byte_out
```

|argv|meaning
|:-:|:-:|
|-f|Original file|
|-w|Writing mode|
|-o|Out file|
